CSS: Styling Basics

Introduction to CSS, how to style HTML elements.

CSS (Cascading Style Sheets) is used to style HTML elements and create visually appealing web pages. Here’s an overview of key concepts:

What are Selectors?

Selectors are used in CSS to target HTML elements that you want to style. Common types of selectors include: Element selectors: Target specific HTML tags, like p {} to style all 'p' tags. Class selectors: Target elements with a specific class attribute, written as .classname {}. ID selectors: Target a unique element with a specific id, written as #idname {}. Attribute selectors: Target elements based on their attributes, such as input[type="text"].

Basic CSS Properties

Color: Used to set text or background color. For example, color: red; changes text color, while background-color: blue; sets the background color. Font size: Controls text size with font-size, such as font-size: 16px;. Margins: Define space outside an element. margin: 10px; sets an equal margin on all sides. Padding: Defines space inside the element, between the content and the element’s border. For example, padding: 15px;. The Importance of Cascade and Specificity CSS follows a cascading order, meaning styles are applied based on their order of appearance. Specificity determines which styles take priority. Inline styles have the highest priority, followed by ID selectors, class selectors, and finally element selectors.

How to Link CSS to HTML

Inline styles: Added directly to the HTML tag using the style attribute. Internal styles: Written within the 'style' tag inside the HTML’s 'head'. External styles: Linked using 'link'to an external CSS file, which is the preferred method for maintaining clean and reusable code. Understanding these key concepts allows developers to efficiently style web pages with CSS.
